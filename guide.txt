■ なにめしって？

「今日なに食べよう？」を解決してくれるAIレシピ提案アプリだよ。
冷蔵庫の食材を入れたら、季節や気分に合ったレシピを提案してくれる！

━━━━━━━━━━━━━━━━━━━━━━━━

■ 基本の使い方

1. ✨ 今日のおすすめ（一番カンタン！）
   ホーム画面の紫のボタンをタップするだけ！
   食材の入力なしで、季節・時間帯・最近の食事を考慮した
   レシピが出てくるよ。

2. 🗣️ 相談する
   「冷蔵庫に鶏肉とキャベツがある」みたいに食材を入れると、
   それに合ったレシピを提案してくれる。
   気分（がっつり/あっさり等）や調理時間も選べる。

3. 🎲 おまかせ
   何も考えたくない日に。ランダムにレシピを提案。

4. 😴 手抜きモード
   包丁なし・5分以内・電子レンジ中心の超簡単レシピだけ！
   疲れた日に。

━━━━━━━━━━━━━━━━━━━━━━━━

■ レシピが出てきたら

・📝 食べた！    → 作った料理を記録。写真も撮れる📸
・🔄 別のレシピ  → 同じ条件で別のレシピを提案
・💾 お気に入り  → また作りたいレシピを保存
・🛒 買い物リスト → 材料をまとめて買い物リストに追加
・📤 シェア      → レシピをLINE等で共有
・💬 質問する    → 「醤油がない場合は？」みたいに聞ける
・⏲️ タイマー    → 調理時間に合わせたタイマー付き

━━━━━━━━━━━━━━━━━━━━━━━━

■ その他の機能

・📖 食事履歴    → カレンダーで記録を確認。写真付きで振り返れる
・✏️ 手動記録    → 外食や自分で考えた料理も写真付きで記録
・🛒 買い物リスト → チェックしながら買い物。お気に入りからも追加できる
・📊 栄養バランス → PFC（タンパク質・脂質・炭水化物）をAIが自動推定
・⚙️ 設定       → 栄養目標の設定、お気に入り管理、ログアウト

━━━━━━━━━━━━━━━━━━━━━━━━

■ 最初にやること

1. URLを開く
2. Googleアカウントでログイン
3. Groq APIキーを入力（だいちゃんに聞いてね）
4. あとは「今日のおすすめ」をタップするだけ！
